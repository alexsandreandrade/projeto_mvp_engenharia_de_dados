# MVP – Engenharia de Dados

Este repositório reúne os artefatos do projeto de Engenharia de Dados (MVP), incluindo:

- Documento em PDF com a descrição completa do projeto
- Arquivo Power BI (.pbix) com o dashboard desenvolvido
- Vídeo de navegação/apresentação do painel

As pastas serão organizadas da seguinte forma:

- `Arquivos de dados .cvs/` → arquivos de fonte de dados originais
- `powerbi/` → arquivo .pbix do dashboard
- `video/` → link para o vídeo de apresentação
